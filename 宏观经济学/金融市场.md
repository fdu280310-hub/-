## 货币供给

银行体系
 - 中央银行 ：政府通过它执行货币政策
   1. 对内：==**职能1**==发行货币+==**职能2**==接受商业银行的存款
   2. 对外：外汇储备
 - 商业银行 ：目标：**放贷**且希望放贷越多越好
     但之前先需要交给中央银行存款准备金
 - 大众：**决定C 和 D的比重**(多少放自己手上，多少存银行)
### 银行体系的运转

1. 货币市场依托于生产市场，所以需要一定收入
2. 两种选择
     1. 大众保留**通货** C： ==**中央银行对大众的债务，体现中央银行的信誉**==
     2. 存入商业银行 D
     分配比例的刻画：**通货保有率** $c = \frac{C}{C+D} = \frac{C}{M}$
3. 商业银行向中央银行缴纳存货款准备金 R (仍属于商业银行) : ==**中央银行对商业银行的债务，体现中央银行的信誉**==
    - 高能货币：H = C + R
#### 两组核心线索

 - 信誉问题 : 2个高能货币（C & R)
 - 资金流向问题 ：2率（通货保有率&存款准备金率）
### 商业银行的放贷机制

1. H = 100，C = 0
     存款准备金 = $S_n$ = $\lim\limits_{n  \to  \infty}$$100\theta*\frac{1}{1-(1-\theta)}$ =$100$
     **存款总量** D= $\frac{100}{\theta}$ = $\frac{H}{\theta}$
     **货币乘数** M&H关系  M = $\frac{1}{\theta}$ H
     - 经济学直觉：$\theta$ → 0 ，银行将无限放贷
 2. H = 100, 0 < C <1
     **存款总量** D= $\frac{100(1-c)}{1-(1-c)(1-\theta)}$ 
     **货币乘数** M&H关系  M = $\frac{1}{1-(1-c)(1-\theta)}$ H 
      - c 与 $\theta$ 均为**外生变量** （模型内无因素影响c与$\theta$，给定的值）
      - c →1 $\theta$ →1 商业银行创造货币机制趋于失败
- 谁在放贷过程当中起到关键性作用？
  1. 公众。正是因为公众参与到存贷之中，才能使放贷链条传下去。**C 反映了公众的作用力**
  2. 中央银行。**$\theta$ 反映了商业银行作用力**
  3. 商业银行→把 c 和 $\theta$ 揉在一起
#### 中央银行如何影响货币供给

货币供给方程：M = $\frac{1}{1-(1-c)(1-\theta)}$ H 
1. 影响基础货币H（多数）**买卖债卷**影响基础货币
     1. 卖出债卷 H↓
     2. 回收债卷 H↑
2. 影响存款准备金$\theta$
